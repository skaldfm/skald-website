Under the hood

Simple stack. No surprises.

Single binary, single file database, single container. Deploy in seconds, maintain with ease.

Go backend

Fast, compiled, no runtime dependencies. One binary is all you need.

SQLite database

Your data in a single file. WAL mode for performance. Automatic backups included.

HTMX + Tailwind

Dynamic UI without JavaScript framework complexity. Server-rendered, progressively enhanced.

Docker ready

One docker compose up -d and you're running. Volume mount for persistent data.
